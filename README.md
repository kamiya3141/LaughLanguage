<h1>・言語使用</h1>


<h3>・"ﾃﾞｭﾌw" -> ポインタの値を1増やす</h3>

<h3>・"ｸﾞﾌｯw" -> ポインタの値を1減らす</h3>

<h3>・"ｱﾋｬw" -> ポインタを右にずらす</h3>

<h3>・"ﾋﾋﾋw" -> ポインタを左にずらす</h3>

<h3>・"wwww" -> ポインタの値を出力する</h3>
